¿Qué es CSS y por qué es útil?

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML.

CSS permite separar la estructura del documento HTML de su presentación visual, lo que hace que el código HTML sea más legible y fácil de mantener.

Con CSS, puedes definir estilos para elementos específicos, como encabezados, párrafos, enlaces y más. Puedes controlar la apariencia de estos elementos, como el color del texto, el tamaño de fuente, el margen y el espaciado entre líneas.
